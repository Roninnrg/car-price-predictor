# Оценка стоимости автомобилей (FastAPI + Streamlit)

Мой учебный проект, в котором я прошел весь путь: от анализа данных до создания рабочего веб-приложения. Программа предсказывает рыночную цену б/у машины на основе ее характеристик (год выпуска, пробег, тип коробки передач и т.д.).

## Что внутри проекта:
* **Анализ данных:** Почистил датасет от сильных выбросов (машин с аномальным пробегом) и перевел текстовые признаки в числа с помощью One-Hot Encoding.
* **Машинное обучение:** Обучил модель Случайного леса (Random Forest Regressor). На тестовой выборке она показала отличную точность — коэффициент R² получился в районе 0.97. Модель сохранена в файл `model.pkl`.
* **Бэкенд (FastAPI):** Скрипт `main.py` запускает сервер, который принимает данные машины по API, передает их в модель и возвращает предсказанную цену.
* **Фронтенд (Streamlit):** Скрипт `app.py` создает простую веб-форму в браузере, чтобы можно было удобно вводить параметры и отправлять запросы на бэкенд.

## Как запустить проект у себя:

1. Установить библиотеки из файла зависимостей:
```bash
pip install -r requirements.txt
```

2. Запустить бэкенд (сервер FastAPI):
```bash
uvicorn main:app --reload
```

3. В новом окне терминала запустить интерфейс (Streamlit):
```bash
streamlit run app.py
```
После этого в браузере откроется страница приложения, где можно протестировать форму.
